# Advent of Code

These are the puzzles and solutions
for [Advent of Code](https://adventofcode.com/).

There is a
[subReddit](https://www.reddit.com/r/adventofcode/wiki/solution_megathreads)
with solutions and people kind enough to help people discover their problems.

## Directories

Directory [`parent`](parent/) contains a POM file to serve as a parent project
for all the Java source directories.

Directory [`common`](common/) contains code that is common to multiple year's
worth of puzzles. These are things like data objects, algorithms, and solution
start-up classes.

Directory [`template`](template/) is a template to copy to quickly create a new
directory tree for a new year's puzzles.

Directories named `20xx` are the puzzles and solutions for the specified year.
